Breast Cancer Diagnosis

Introduction

In this project, we aim to develop a machine learning model capable of predicting breast cancer diagnosis based on patient data. Breast cancer is one of the most common cancers worldwide, and early detection is critical for improving survival rates. Using a dataset of diagnostic measurements, we will build a neural network to classify tumors as either benign or malignant.
The project will guide you through the complete data science pipeline, including data preprocessing, model building, training, and evaluation. By the end, we will demonstrate the model’s effectiveness and discuss its potential applications in medical diagnostics, showcasing the power of machine learning in contributing to healthcare advancements.

Pipeline

After setting up our environment, we load the breast cancer dataset into a pandas DataFrame. This dataset contains various diagnostic measurements used to predict breast cancer 


import pandas as pd
from matplotlib import pyplot as plt
import seaborn as sns
import numpy as np
										

df = pd.read_csv(r"..../breast_cancer_dataset.csv")
print(df.describe().T)

Here we focus on cleaning and preparing the data for analysis. We remove any rows containing missing values. Then, we check the data types of each column to ensure they are correctly formatted 



	
df = df.dropna()
										
									


	
df = df.rename(columns={'Diagnosis':'Label'})
print(df.dtypes)

Here, we visualize the distribution of the diagnosis labels in our dataset. This chart shows us the frequency of malignant (M) and benign (B) cases, giving us a quick overview of how balanced our dataset is. 


sns.countplot(x="Label", data=df)

In this step, we focus on preparing the data for machine learning. First, we examine the distribution of the diagnosis labels to understand the balance between malignant and benign cases. We then encode the labels into numerical values. Next, we prepare the feature set 'x' 


print("Distribution of data: ", df['Label'].value_counts())

y = df["Label"].values
print("Labels before encoding are: ", np.unique(y))

from sklearn.preprocessing import LabelEncoder
labelencoder = LabelEncoder()
Y = labelencoder.fit_transform(y) # M=1 and B=0
print("Labels after encoding are: ", np.unique(Y))
									

X = df.drop(labels = ["Label", "ID"], axis=1) 
print(X.describe().T)

In this step, we normalize the feature data to ensure that all the features are on the same scale 


from sklearn.preprocessing import MinMaxScaler
scaler = MinMaxScaler()
scaler.fit(X)
X = scaler.transform(X)
print(X)

We split the dataset into training and testing sets. 


from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(X, Y, test_size=0.25, random_state=42)
print("Shape of training data is: ", X_train.shape)
print("Shape of testing data is: ", X_test.shape)

We build a neural network model using Keras. The first layer is a Dense layer with 16 neurons and a ReLU activation function, which will help the model learn complex patterns. We then add a Dropout layer to prevent overfitting by randomly dropping 20% of the neurons during training. 


from keras.models import Sequential
from keras.layers import Dense, Activation, Dropout
model = Sequential()
model.add(Dense(16, input_dim=30, activation='relu')) 
model.add(Dropout(0.2))
model.add(Dense(1)) 
model.add(Activation('sigmoid')) 
 
model.compile(loss='binary_crossentropy', optimizer='adam', metrics=['accuracy'])

print(model.summary())

In this step, we train the neural network model using the training data. After training, we visualize the training and validation loss as well as accuracy over the epochs. The first plot shows how the loss decreases as the model learns, while the second plot illustrates how the accuracy improves over time 


history = model.fit(X_train, y_train, verbose=1, epochs=100, batch_size=64,
		validation_data=(X_test, y_test))
	

loss = history.history['loss']
val_loss = history.history['val_loss']
epochs = range(1, len(loss) + 1)
plt.plot(epochs, loss, 'y', label='Training loss')
plt.plot(epochs, val_loss, 'r', label='Validation loss')
plt.title('Training and validation loss')
plt.xlabel('Epochs')
plt.ylabel('Loss')
plt.legend()
plt.show()


acc = history.history['accuracy']  #Use accuracy if acc doesn't work
val_acc = history.history['val_accuracy']  #Use val_accuracy if acc doesn't work
plt.plot(epochs, acc, 'y', label='Training acc')
plt.plot(epochs, val_acc, 'r', label='Validation acc')
plt.title('Training and validation accuracy')
plt.xlabel('Epochs')
plt.ylabel('Accuracy')
plt.legend()
plt.show()

In the final step, we evaluate the model's performance on the test data. Next, we calculate the confusion matrix, which provides a detailed breakdown of the model's classification results 


y_pred = model.predict(X_test)
y_pred = (y_pred > 0.5)

from sklearn.metrics import confusion_matrix
cm = confusion_matrix(y_test, y_pred)

sns.heatmap(cm, annot=True)

Conclusion

Throughout the training process, we monitored the model's performance, ensuring it learned effectively without overfitting. The final evaluation using a confusion matrix demonstrated the model’s strong predictive capabilities, accurately distinguishing between benign and malignant cases. This project highlights the potential of neural networks in medical diagnostics, offering a glimpse into how such models can assist in early detection and treatment planning for breast cancer, ultimately contributing to better patient outcomes.
